# Python Interactive Calculator

This is a simple, object-oriented Python terminal application that makes everyday basic calculations easy. 

## Features   Of The Calculator
* **Core Math Operations:** Performs addition, subtraction, multiplication, and division.
* **Smart Error Handling:** Robustly catches inputs that aren't numbers and prevents system crashes if a user tries to divide by zero(Divisible by Zero Mathematical Error)
* **Interactive Loop:** Includes a welcoming interface that asks the user if they want to continue performing calculations or exit.

## Tech Stack
* **Language:** Python 3.x
* **Architecture:** Object-Oriented Programming (built using a `Calculator` class)

## How to Run the Project

1. **Ensure you have Python installed** on your local machine.
2. **Download or copy** the script file (e.g., `calculator.py`) into your project folder.
3. **Open your terminal** or command prompt, navigate to the folder, and run:
   ```bash
   python calculator.py
